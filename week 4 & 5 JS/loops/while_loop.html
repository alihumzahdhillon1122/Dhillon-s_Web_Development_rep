The syntax of a while loop is straightforward


while condition: # Block of code to be executed repeatedly
  1. The condition is evaluated before each iteration of the loop. If the condition is true, the block of code inside the loop is executed. If the condition is false, the loop terminates, and the program continues with the code after the loop.
  2. Inside the loop, you typically perform some operations that may change the value of variables used in the condition. This allows the condition to eventually become false and exit the loop.

let i = 1; while (i <= 5) { console.log(i); i++; }
