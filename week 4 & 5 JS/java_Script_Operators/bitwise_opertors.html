Bitwise Operators

this topic was out of my mind

Operator Description Example
& Bitwise AND: Sets each bit to 1 if both bits are 1. x & y;
| Bitwise OR: Sets each bit to 1 if either bit is 1. x | y;
^ Bitwise XOR: Sets each bit to 1 if only one of the bits is 1. x ^ y;
~ Bitwise NOT: Inverts all the bits. ~x;
<< Left Shift: Shifts the bits of x to the left by y positions. x << y;
>> Right Shift: Shifts the bits of x to the right by y positions. x >> y;
