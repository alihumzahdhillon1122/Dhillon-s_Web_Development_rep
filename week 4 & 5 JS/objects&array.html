purpose

  • Arrays: Arrays are used to store ordered collections of data. They are typically used when you have a list of items that you want to access by their index.
  • Objects: Objects are used to store collections of key-value pairs, where each key is unique. They are used to represent structured data and organize related information.

synatx

  • Arrays: Arrays are created using square brackets [] and contain elements separated by commas.
  • Objects: Objects are created using curly braces {} and contain key-value pairs separated by commas, with each key and value separated by a colon :.
